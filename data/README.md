# HealthPilot Data Directory

## Sample Data (`sample/`)

Contains synthetic health data mimicking Google Fit CSV exports. Generated by `generate_sample_data.py`.

### Files

**daily_activity.csv** — Daily step count, calories, distance, and active minutes.

| Column | Type | Description |
|--------|------|-------------|
| date | YYYY-MM-DD | Calendar date |
| steps | int | Total daily steps |
| calories_burned | int | Total calories burned (BMR + activity) |
| distance_km | float | Total distance walked/run |
| active_minutes | int | Minutes of moderate+ activity |

**heart_rate.csv** — Heart rate measurements at 5-minute intervals.

| Column | Type | Description |
|--------|------|-------------|
| timestamp | ISO 8601 | Measurement time |
| bpm | int | Beats per minute |

**sleep.csv** — Nightly sleep sessions.

| Column | Type | Description |
|--------|------|-------------|
| date | YYYY-MM-DD | Night the sleep started |
| sleep_start | HH:MM | Bedtime |
| sleep_end | HH:MM | Wake time |
| duration_hours | float | Total sleep duration |
| deep_sleep_pct | float | % in deep sleep |
| light_sleep_pct | float | % in light sleep |
| rem_pct | float | % in REM sleep |

## User Profile (`user_profile.json`)

Stores user preferences, dietary restrictions, fitness goals, and targets. Read by agents to personalize recommendations.

## How to Export Your Own Google Fit Data

1. Go to [Google Takeout](https://takeout.google.com/)
2. Select only "Fit" data
3. Export as CSV format
4. Place the exported files in `sample/` with the column names above

See `docs/DATA_GUIDE.md` for detailed instructions on mapping Google Fit columns to this schema.
